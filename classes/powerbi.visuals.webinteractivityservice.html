Private syncSelectionState

  • syncSelectionState(filterPropertyId?: DataViewObjectPropertyIdentifier): void
    • Defined in services/interactivityService.ts:278

    Syncs the selection state for all data points that have the same category. Returns true if the selection state was out of sync and corrections were made; false if the data is already in sync with the service.

    If the data is not compatible with the current service's current selection state, the state is cleared and the cleared selection is sent to the host.

    Ignores series for now, since we don't support series selection at the moment.

    Parameters

    • Optional filterPropertyId: DataViewObjectPropertyIdentifier

    Returns void
