  • addNode(item: any, items: any, rowIndex: number, depth: number): ITablixCell
    • Defined in controls/tablix/tablixDimension.ts:240

    This method can be thought of as the continuation of addNodes() as it continues the DFS (Depth First Search) started from addNodes(). This function also handles ending the recursion with "_needsToRealize" being set to false.

    Once the body cells are reached, populating is done linearly with addBodyCells().
